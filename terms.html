2. Description of the App

2.1 Core Functionality

BlessUp is a faith-based daily prayer application that provides the following services:

  • Daily Prayer Flow: Guided prayer sessions consisting of a morning prayer, daily scripture reading, and gratitude reflection.
  • App Shielding: Integration with Apple's Screen Time API and Family Controls framework to temporarily restrict access to user-selected applications until the daily prayer flow has been completed.
  • Streak Tracking: Tracking and display of consecutive daily prayer completion, including milestones and historical data.
  • Scripture Library: Curated collection of Bible verses organized by category, delivered through daily rotation.
  • Widgets: Home Screen and Lock Screen widgets displaying streak data, daily verses, and prayer status.
  • Local Notifications: Customizable daily prayer reminders and periodic progress summaries.

2.2 Screen Time and App Shielding

The App uses Apple's Screen Time API, Family Controls framework, Device Activity framework, and Managed Settings framework to provide app-shielding functionality. By granting Screen Time permissions, you authorize the App to:

  • Apply shields to applications and application categories that you have selected;
  • Remove shields upon successful completion of your daily prayer flow;
  • Reset shields daily at midnight via a Device Activity Monitor extension;
  • Display a custom shield interface with a prompt to begin your daily prayer.

You understand and acknowledge that this functionality requires explicit authorization through Apple's Screen Time permission system and that you may revoke this authorization at any time through your device's Settings > Screen Time > BlessUp.
